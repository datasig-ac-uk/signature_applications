## Installation

1. Follow the instructions on [esig repo](https://github.com/datasig-ac-uk/esig) to install the `esig` Python package.

2. Install `jupyterlab`:

```bash
pip install jupyterlab
```

3. Open notebooks by:

```bash
jupyter notebook
```

Alternatively, you can start JupyterLab by:

```bash
jupyter lab
```
